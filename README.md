🌐 Simple Portfolio Website
This is a basic personal portfolio website created using HTML and CSS. It serves as a beginner-friendly project to showcase personal information, skills, and a simple layout structure.

✨ Features:
Built using plain HTML and basic CSS

Sections include: About Me, Skills, Projects, and Contact

Simple layout suitable for desktop viewing

Great for beginners learning web development fundamentals

⚠️ Note:
This version is not responsive and is best viewed on a desktop screen. Future updates may include mobile responsiveness and layout enhancements.

🚀 How to Use:
Clone or download the repository

Open in your browser to view the site
